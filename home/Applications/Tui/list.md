# All Tui programs I am using and their following updating/installing methods

## bluetui
https://github.com/pythops/bluetui
install - "cargo install bluetui"
update - "cargo install bluetui --force"

## wiremix
https://github.com/tsowell/wiremix
install - "sudo dnf install wiremix" 
update - "sudo dnf upgrade"

## nmtui-go
https://github.com/doeixd/nmtui-go
install - install from github releases
update - exec "$HOME/Applications/Tui/nmtui-go" --update

## clipse
https://github.com/savedra1/clipse
install - "sudo dnf copr enable azandure/clipse && sudo dnf install clipse"
update - "sudo dnf upgrade"
